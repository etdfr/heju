    顾锦和：“去后院。”

    先前一起办事时他来过几次，每次谢衍都是在后院忙碌。

    他也好奇谢衍藏了什么宝贝在后院。

    顾锦和随意应了一句，目光却紧紧盯着眼前一个个大棚。
